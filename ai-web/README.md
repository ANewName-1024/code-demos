# NeuroNex AI Website

A premium AI company website with futuristic design.

## Live Demo

**URL:** http://8.137.116.121:8443

## Features

- 🎨 Modern dark theme with cyan/purple gradient
- 🧠 Neural network animated background
- 💬 Interactive AI demo terminal
- 📊 Animated statistics counters
- 📱 Fully responsive design

## Tech Stack

- Pure HTML/CSS/JS
- Three.js for 3D effects
- Font Awesome icons
- Google Fonts

## Deployment

```bash
# Deploy to server
scp -r . user@server:/var/www/ai-web/

# Configure Nginx on port 8443
```
